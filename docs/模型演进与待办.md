## 2. 历史教训（简）

- **Init 死锁**：`Init` 持锁内调 `EnableSlot` → 已改为锁外 `Clone+Init`。
- **运行期卡顿**：探针 + ppocr 每帧抢 NPU → 已通过移除 OCR 消除。
- **Ctrl+C**：二次 SIGINT `_exit`；Stop 时队列 drain + quit 哨兵。
